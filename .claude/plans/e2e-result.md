## 발견된 이슈

### Issue 1: 캔버스 PointerEvent setPointerCapture 에러
- **심각도**: warning
- **재현 방법**: DrawingCanvas에서 커스텀 PointerEvent 디스패치 시 발생
- **원인**: `DrawingCanvas.tsx:109`의 `e.target.setPointerCapture(e.pointerId)` 호출이 시뮬레이션된 이벤트에서는 실제 포인터 ID가 없어 실패
- **영향**: E2E 자동화 테스트에서 드로잉 시뮬레이션 불가 (실제 사용자 사용에는 영향 없음)
- **해결 방향**: try-catch로 setPointerCapture 감싸거나, 테스트용 mock 이벤트 지원

### Issue 2: 결과 화면 "끝!" 버튼 영구 비활성화
- **심각도**: critical
- **재현 방법**: 2명 플레이어로 게임 완료 후 결과 화면에서 마지막 스토리(2/2)로 이동
- **증상**: `button "끝!" [disabled]` — 마지막 이야기에서도 끝내기 버튼 클릭 불가
- **영향**: 게임 종료 후 로비로 돌아갈 수 없음 (새로고침 필요)
- **예상 원인**: 결과 화면의 "끝" 버튼 활성화 조건이 잘못됨 (모든 플레이어가 결과를 확인했는지 여부로 체크하는 것으로 추정)

### Issue 3: 결과 화면 체인 표시 불완전
- **심각도**: info
- **증상**: 원래 주제만 표시되고 그림→추측 체인 과정이 표시되지 않음
- **원인**: 타이머 만료로 인해 빈 그림이 제출되어 체인 스텝이 1개뿐
- **비고**: 정상 플레이 시에는 전체 체인이 표시될 것으로 예상
